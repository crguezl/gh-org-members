## Installation

```
gh extension install crguezl/gh-org-members
```

## Help

```
➜  github-profile-readme gh help org-members
Usage: gh org-members [options] [organization] [options]

Options:
  -V, --version          output the version number
  -f, --fullname         show name of the user (if filled)
  -j, --json             returns the full json object
  -r, --regexp <regexp>  filter <query> results using <regexp>
  -o --org <org>         default organization or user
  -h, --help             display help for command

  - You can set the default organization through the GITHUB_ORG environment variable
```
